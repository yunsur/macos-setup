# Homebrew Cask

> Homebrew Cask 是 Homebrew 的 macOS Native 应用扩展，通过 cask 可以安装各类应用程序

## Homebrew 用法
> 搜索 Cask 的方法和搜索普通包一样，但安装时需要加上 `--cask` 指令:

* 搜索

```bash
brew search <formula>
```

* 安装

```bash
brew install --cask <formula>
```

* 建议

```bash
brew install --cask \
    alfred \
    android-file-transfer \
    appcleaner \
    caffeine \
    cheatsheet \
    colloquy \
    docker \
    doubletwist \
    dropbox \
    google-chrome \
    google-hangouts \
    flux \
    1password \
    rectangle \
    sublime-text \
    superduper \
    transmission \
    valentina-studio \
    vlc
```
