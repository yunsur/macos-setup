# 系统偏好设置
> macOS 系统偏好设置

## 通用

* 外观 > 自动

## 桌面与屏幕保护程序

* 触发角
    * 左上 > 无动作
    * 左下 > 启动台
    * 右上 > 无动作
    * 右下 > 桌面


## 聚焦

* 搜索结果
    * 关闭 > 字体
    * 关闭 > 书签与历史记录


## 用户与群组

* 密码
    * 允许 > 用户使用 Apple ID 重设密码

* 登录选项
    * 将快速用户切换菜单显示为 > 图标

## 辅助功能

* 指针控制
    * 鼠标与触控板 > 触控板选项 > 启用拖移 > 三指拖移


## 安全性与隐私

* 允许任何来源
    ```bash
    sudo spctl --master-disable
    ```

* 关闭任何来源
    ```bash
    sudo spctl --master-enable
    ```

## 键盘
* 键盘
    * 修饰键
        * 大写锁定(⇪)键 > ⌃ Control
        * Control(⌃)键 > ⇪ 大写锁定键
* 输入法
    * 简体双拼 > 小鹤双拼


## 触控板

* 光标与点按
    * 查询与数据检测 > 三指轻点
    * 辅助点按 > 双指点按或轻点
    * 轻点来点按 > 单指轻点   

* 滚动缩放
    * 默认全选

* 更多手势
    * 默认全选
